## Overview

Electron Apps Showcase is the official app showcase for Electron framework. With 100+ featured apps including VS Code, Slack, and Discord, it demonstrates cross-platform desktop app development using JavaScript, HTML, and CSS.

## Key Statistics

- **100+ featured apps** (estimated)
- Cross-platform support (Windows, macOS, Linux)
- JavaScript/HTML/CSS development
- OpenJS Foundation project

## Famous Electron Apps

- **Visual Studio Code** - Code editor
- **Slack** - Team communication
- **Discord** - Gaming chat
- **Postman** - API testing
- **WhatsApp Desktop** - Messaging
- **Notion** - Productivity

## Electron Fiddle

The easiest way to get started:
- Quick-start templates
- Monaco Editor (VS Code)
- Binary packaging
- Version selection

## Recent Updates (2024-2025)

- Electron v37.3 release (Nov 2025)
- GSoC 2025 mentoring organization
- API History documentation (GSoC 2024)
- Continuous development

## Development

Build apps using:
- JavaScript
- Node.js
- HTML/CSS
- Electron APIs
- electron-forge packaging

## Not a Traditional Marketplace

Note: Electron Apps is a showcase, not a distribution marketplace. Apps are distributed through their own channels (websites, app stores, etc.).
