## Overview

Arcadier Plugin Marketplace is the official plugin store for Arcadier marketplace platform. With 50+ plugins for business, marketing, and e-commerce, it extends online marketplaces for 10,000+ marketplace operators in 170+ countries.

## Key Statistics

- **50+ plugins** (estimated)
- **10,000+ marketplaces** powered
- **170+ countries** worldwide
- Launched September 2019

## Launch Plugins (2019)

Initial 10 plugins included:
- Marketplace Analytics & Reports
- MailChimp Exporter
- Discount Coupon Generator
- Membership & Subscriptions
- Blog Page Builder
- Web Page Builder
- And more...

## Plugin Types

### Public Plugins
Available for any Arcadier marketplace:
- Submitted to Arcadier
- Approved for marketplace
- Available to all operators

### Private Plugins
For specific marketplaces:
- Custom development
- Limited access
- Tailored functionality

## Plugin Categories

- Business tools
- Marketing
- E-commerce
- Analytics
- CRM integration
- Payment processing

## Development

Build plugins using:
- Arcadier APIs
- Developer platform
- JavaScript SDK
- Plugin guidelines
- Approval process
