## Overview

Grafana Plugins is the official catalog for extending Grafana's observability and visualization platform. With hundreds of plugins built by Grafana Labs, commercial partners, and the community, it extends Grafana's capabilities.

## Plugin Types

### Panel Plugins
Create custom visualizations for any data.

### Data Source Plugins
Connect to databases, APIs, and services.

### App Plugins
Bundle panels, data sources, and pages into cohesive experiences.

## Popular Data Sources

- Prometheus
- InfluxDB
- Elasticsearch
- MySQL/PostgreSQL
- CloudWatch
- Azure Monitor
- Google Cloud Monitoring

## Development Tools

### @grafana/create-plugin
CLI tool for scaffolding, developing, and testing plugins.

### Plugin Tools
- TypeScript/React support
- Grafana UI components
- Test utilities
- Migration tools

## Grafana Cloud

Amazon Managed Grafana supports up to 50 additional plugins per workspace.

## Development

Build plugins using:
- Grafana Plugin Tools
- TypeScript and React
- Go for backend plugins
- Grafana API
- Plugin SDK
