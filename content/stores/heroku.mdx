## Overview

Heroku Elements is the marketplace for Heroku's cloud platform. With 175-200+ managed add-on services, 7,200+ Heroku Buttons, and 7,800+ buildpacks, it extends Heroku's PaaS capabilities.

## Key Statistics

- **175-200+ add-ons** (managed services)
- **7,200+ Heroku Buttons** (one-click deploys)
- **7,800+ buildpacks** available
- Thousands of total elements
- Regular new add-on additions

## Element Types

### Add-ons (175-200+)
Fully managed cloud services:
- Databases (Postgres, Redis, MongoDB)
- Monitoring & logging
- Email & communication
- Search engines
- Caching
- Analytics

### Heroku Buttons (7,200+)
One-click app deployment:
- Template apps
- Starter kits
- Open source projects

### Buildpacks (7,800+)
Language and framework support:
- Official buildpacks
- Community buildpacks
- Custom buildpacks

## Development

Build add-ons using:
- Heroku Add-on Provider API
- Provisioning API
- SSO integration
- Heroku CLI plugins
- Partner portal
