## Overview

Trello Power-Ups marketplace launched in 2016 and offers 200+ integrations that extend Trello's Kanban-style project management boards. As part of Atlassian since 2017, it provides developers access to Trello's millions of active users.

## Key Features

- **200+ Power-Ups** available
- **Unlimited Power-Ups** for all Trello users (including free tier)
- **No commission** - developers handle monetization
- **Part of Atlassian** ecosystem

## Power-Up Categories

- Automation & workflows
- Reporting & analytics
- Time tracking
- CRM & sales integrations
- Development tools
- Communication tools

## Development

Power-Ups are built with:
- JavaScript (browser-based)
- Trello REST API
- Power-Up Client Library
- OAuth for authentication

## Free vs Premium Power-Ups

- Most Power-Ups are free to use
- Some partners offer premium subscriptions
- Premium billing handled by developers
- No Trello-managed payment system
