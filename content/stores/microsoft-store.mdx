## Overview

Microsoft Store is the built-in app store for Windows 10 and 11. In recent years, Microsoft has significantly improved the platform, reducing commission rates and opening it to more app types including traditional Win32 apps and PWAs.

## App Types Supported

### Universal Windows Platform (UWP)
Native Windows apps that can run across Windows 10/11 devices.

### Win32 Apps
Traditional Windows desktop applications packaged in MSIX format.

### Progressive Web Apps (PWAs)
Web apps that can be installed and run like native apps.

### Android Apps (via WSA)
Select Android apps can run on Windows 11 through Windows Subsystem for Android.
