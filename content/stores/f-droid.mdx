## Overview

F-Droid is a non-profit volunteer project that provides a repository of only free and open-source Android apps. Apps are built from source code and verified for transparency.

## Requirements

### Open Source Requirements
- Complete source code must be available
- Must use only FOSS dependencies
- No proprietary Google services
- No tracking or analytics

### Anti-Features
F-Droid labels apps with "anti-features" to inform users about:
- Ads
- Tracking
- Non-free network services
- Non-free dependencies
- Promoting non-free add-ons
