## Overview

AgentExchange is Salesforce's dedicated marketplace for AI agents built on the Agentforce platform. Launched as an evolution of AppExchange for the agentic AI era, it provides a trusted hub for enterprises to discover, evaluate, and deploy pre-built AI agent solutions.

## AgentExchange Components

### Actions
Expand agent capabilities with:
- Custom integrations (Apex, Flows, APIs)
- External service connections
- Data retrieval and manipulation
- Process automation

### Prompt Templates
Pre-written guides for consistent agent behavior:
- Response formatting
- Task-specific instructions
- Brand voice guidelines
- Compliance requirements

### Topics
Focus agent behavior on specific tasks:
- Scope definition
- Guardrail configuration
- Capability boundaries
- Performance optimization

### Agent Templates
Complete AI solutions including:
- Pre-configured topics
- Bundled actions
- Ready-to-deploy agents
- Industry-specific solutions

## Einstein Trust Layer

All agents operate within Salesforce's Einstein Trust Layer:
- **Dynamic Grounding**: Real-time data context
- **Zero Data Retention**: No customer data stored by LLMs
- **Toxicity Detection**: Content safety filtering
- **Audit Trails**: Complete action logging
- **Data Masking**: Sensitive information protection

## Launch Partners

200+ partners including:
- Technology integrators
- ISV partners
- System integrators
- Consulting firms

Over 1,000 partners trained to build AI agents.

## Agentforce Platform

### Agent Builder
Create agents using:
- Low-code/no-code tools
- Natural language instructions
- Pre-built templates
- Custom development

### Agent Capabilities
- **Sales Agents**: Lead qualification, opportunity management
- **Service Agents**: Case resolution, customer support
- **Marketing Agents**: Campaign optimization, personalization
- **Commerce Agents**: Product recommendations, order management

### Multi-Cloud Support
Agents work across:
- Sales Cloud
- Service Cloud
- Marketing Cloud
- Commerce Cloud
- Platform

- Penetration testing
