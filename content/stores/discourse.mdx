## Overview

Discourse Plugins is the official extension ecosystem for Discourse's modern, open-source forum software. With 100+ official and third-party plugins, it enables community administrators to customize and extend their forums.

## Key Statistics

- **100+ plugins** available
- Official directory at discourse.org/plugins
- GitHub all-the-plugins repository
- Active Meta community

## Plugin Sources

### Official Directory
discourse.org/plugins - curated plugins

### Meta Discourse
