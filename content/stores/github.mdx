## Overview

GitHub Marketplace launched in 2017 and eliminated its 25% commission in 2019, making it one of the most developer-friendly marketplaces. It hosts GitHub Apps, Actions, and integrations for the world's largest developer community.

## App Types

### GitHub Apps
- Full OAuth and webhook integrations
- Access to GitHub APIs
- Per-repository or organization-wide installation

### GitHub Actions
- CI/CD workflow automation
- Reusable workflow components
- Marketplace distribution

## Pricing Models

- **Free**: No charge to users
- **Flat rate**: Fixed monthly/yearly fee
- **Per-unit**: Price per user or organization seat
- **Free trial**: Optional 14-day trial period

## Publisher Requirements

Paid plans require verified publisher status:
- Verified email and identity
- Valid payment information
- Compliance with GitHub's policies
