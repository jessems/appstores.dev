## Overview

The Dify Marketplace provides plugins and extensions for Dify, an open-source platform for building AI applications. With visual workflow design, RAG pipelines, and agent capabilities, Dify serves as a comprehensive platform for AI development, and the marketplace extends its functionality.

## Dify Platform Features

### Workflow Builder
Visual canvas for building AI workflows:
- Drag-and-drop interface
- Node-based logic
- Branching and loops
- Testing and debugging

### Model Support
Integration with hundreds of LLMs:
- OpenAI GPT models
- Anthropic Claude
- Google Gemini
- Open-source models (Llama, Mistral)
- Any OpenAI-compatible API

### RAG Pipeline
Complete document processing:
- PDF, PPT, document ingestion
- Text extraction and chunking
- Vector embedding
- Retrieval optimization

### Agent Capabilities
50+ built-in tools including:
- Google Search
- DALL-E
- Stable Diffusion
- WolframAlpha
- Custom API calls
