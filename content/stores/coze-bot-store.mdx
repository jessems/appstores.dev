## Overview

Coze is ByteDance's AI bot development platform, launched in early 2024 as the company's answer to OpenAI's GPTs. It provides a comprehensive no-code environment for building, deploying, and sharing AI chatbots across multiple platforms.

## Key Features

### Bot Builder
The drag-and-drop interface enables:
- Natural language bot configuration
- Visual workflow design
- Plugin integration
- Knowledge base management

### 60+ Built-in Plugins
Plugins cover various use cases:
- Information reading and web search
- Travel planning
- Productivity tools (PPT generation, mind maps)
- Image understanding
- Audio/video processing
- API integrations

### Multi-Channel Deployment
Deploy bots to:
- Discord
- Telegram
- Slack
- Facebook Messenger
- Reddit
- Line
- WhatsApp (coming)

## Technical Architecture

### Doubao Model
Coze runs on ByteDance's in-house Doubao large language model, with support for:
- Text generation
- Image understanding
- Code generation
- Multimodal interactions

### MCP Support
Coze supports the Model Context Protocol (MCP) for:
- Integration with Lark (Feishu)
- Docs and Sheets automation
- External service connections

### Coze Studio (Open Source)
The core engine is available as open-source:
- GitHub: github.com/coze-dev/coze-studio
- One-stop visual development for AI Agents
- Self-hosting options

## Coze Space (2025)

ByteDance launched Coze Space in 2025, an internal beta for AI agent collaboration:
- General-purpose agent interface
- "Expert Agent" system for specialized tasks
- Team collaboration features

## Getting Started

1. Sign up at coze.com (international) or coze.cn (China)
2. Create a new bot in the Bot Builder
3. Configure personality and knowledge
4. Add plugins for extended capabilities
5. Test in the playground
6. Deploy to your preferred channels
