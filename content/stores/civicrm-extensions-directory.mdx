## Overview

CiviCRM Extensions Directory is the official extension repository for CiviCRM open source nonprofit CRM. With 200+ extensions for fundraising, events, memberships, and communications, it extends CiviCRM for nonprofits, NGOs, and civic organizations worldwide.

## Key Statistics

- **200+ extensions** (estimated)
- Open source since 2005
- Multi-CMS support (WordPress, Drupal, Joomla, Backdrop)
- Usage tracking across thousands of sites

## Popular Extensions (by usage)

- **Mosaico** - Email template builder (4,007 sites)
- **Extended Reports** - Advanced reporting (2,442 sites)
- **CiviRules** - Automation rules (2,386 sites)

## Recent Changes (2024)

### CiviCRM 5.79 (November 2024)
- Updated in-app extension directory
- Administrators can now install any stable extension
- Previously limited to reviewed extensions only
