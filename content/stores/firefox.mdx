## Overview

Firefox Add-ons (addons.mozilla.org, or AMO) is Mozilla's official marketplace for Firefox browser extensions, themes, and add-ons. With over 25,000 extensions and no fees or commissions, it's one of the most developer-friendly browser extension platforms.

## Key Statistics

- **25,000+ extensions** available
- **40% of Firefox users** have at least one add-on installed
- **1,000+ Android extensions** (launched December 2023, reached 1,000 by May 2024)
- Usage varies by country: 19% in India, 60% in Russia

## WebExtensions API

Firefox uses the WebExtensions API, which is largely compatible with Chrome:
- Similar manifest structure
- Most Chrome extensions can be ported
- Some Firefox-specific APIs available
- No Manifest V3 restrictions (more developer freedom)

## Firefox for Android

Mozilla launched an open ecosystem for Android extensions in December 2023:
- Started with 400+ extensions
- Grew to 1,000+ by May 2024
- Compatible with Firefox mobile browser

## Review Process

All add-ons undergo review:
- Automated scanning for known issues
- Human review by Mozilla team
- Code must be readable (no obfuscation)
- Typically 1-5 days for review
