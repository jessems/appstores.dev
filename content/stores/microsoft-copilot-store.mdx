## Overview

The Microsoft 365 Copilot Store is the official marketplace for extending Microsoft's AI-powered productivity assistant. Plugins and agents appear in Teams, Outlook, Word, Excel, PowerPoint, and the dedicated Agent Store within Microsoft 365 Copilot.

## Extensibility Options

### Agents
Custom AI agents built with:
- **Copilot Studio**: No-code agent builder
- **Azure AI Foundry**: Advanced AI development
- **Visual Studio Code**: Full code control

### API Plugins
Connect Copilot to external services:
- REST API integrations
- OpenAPI specification support
- Authentication handling

### Copilot Connectors
Bring external data into Copilot:
- Data source connections
- Microsoft Graph integration
- Enterprise data access

## Development Tools

### Microsoft 365 Agents SDK
- Core agent components
- Enterprise-grade publishing
- Multi-channel deployment (Teams, web, 15+ channels)

### Teams Toolkit
- Visual Studio Code extension
- Quick plugin scaffolding
- Local debugging support

### Copilot Studio
- No-code/low-code builder
- Visual conversation designer
- Built-in testing

## Publishing Process

1. **Develop**: Build using SDK, Teams Toolkit, or Copilot Studio
2. **Test**: Local testing and validation
3. **Submit**: Upload to Microsoft Partner Center
4. **Certify**: Pass security and policy review
5. **Publish**: Deploy to marketplace
6. **Enable**: IT admins enable for organization

## Enterprise Features

- **Admin Controls**: IT admins control which plugins users can access
- **Security**: Enterprise-grade security certification
- **Compliance**: Meet organizational compliance requirements
- **Governance**: Audit logs and usage tracking

## Monetization

Sell through Microsoft Commercial Marketplace:
- Per-seat licensing
- Usage-based pricing
- Subscription models
- Enterprise agreements
