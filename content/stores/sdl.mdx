## Overview

RWS AppStore (formerly SDL AppStore) is the official plugin marketplace for Trados Studio translation software. With 200+ plugins including 80+ open-source community plugins on GitHub, it extends translation management with machine translation, productivity tools, and integrations.

- **200+ plugins** (estimated)
- **80+ open-source plugins** on GitHub
- **80+ MT providers** supported
- Industry-leading translation CAT tool
