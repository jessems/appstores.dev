## Overview

Twitch Extensions launched in 2017 and offers 200+ interactive add-ons for Twitch's live streaming platform. Extensions enhance streams with overlays, panels, and interactive features that engage viewers.

## Key Statistics

- **200+ extensions** available
- New extensions added monthly
- ~200 extensions in Viewer Engagement category alone
- 6 extension slots per channel

## Extension Slots

Streamers can use up to 6 extensions:
- Up to 2 component extensions
- Maximum 1 overlay extension
- Maximum 3 panel extensions

## Extension Types

### Overlay Extensions
Full-screen transparent overlays on video.

### Panel Extensions
Displayed below the video player.

### Component Extensions
Positioned anywhere on the video.

## Popular Categories

- Viewer engagement
- Polls & voting
- Leaderboards
- Loyalty & rewards
- Sound alerts
- Game integration
- Analytics

## Monetization

### Bits Integration
Extensions can accept Bits (Twitch's virtual currency):
- Viewers spend Bits for interactions
- Developers earn revenue share
- ~80% to developer (estimated)

## Development

Build extensions using:
- Twitch Developer Console
- Extension Helper (JavaScript)
- React/TypeScript
- Twitch API integration
- PubSub for real-time updates
