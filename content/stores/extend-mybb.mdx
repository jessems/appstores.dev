## Overview

Extend MyBB is the official plugin and theme repository for MyBB forum software. With 2,400+ plugins and themes including alerts, shoutboxes, profile features, and moderation tools, it extends MyBB forums with community-contributed resources.

## Key Statistics

- **2,400+ plugins and themes**
- **Hundreds of modifications** available
- Open source since 2002
- Active community development
