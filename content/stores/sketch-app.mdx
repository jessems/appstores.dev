## Overview

Sketch Plugins is the official extension ecosystem for Sketch's professional vector design tool. With hundreds of plugins, it extends Sketch's capabilities for UI/UX designers at companies like Facebook, Google, Porsche, and Xbox.

## Key Statistics

- **Hundreds of plugins** available
- **300,000+ users** for plugins like Stark
- Active GitHub plugin directory
- Regular updates through 2025

## Plugin Distribution

### Official Directory
sketch.com/extensions/plugins/ - curated plugins

### GitHub Directory
github.com/sketchplugins/plugin-directory - community maintained

Auto-publishing tool for developers
