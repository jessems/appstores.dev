## Overview

Joomla Extensions Directory (JED) is the official marketplace for Joomla CMS extensions. With approximately 6,000 extensions, it's the largest repository of Joomla components, modules, plugins, and templates.

## Key Statistics

- **~6,000 extensions** available
- **5 extension types** supported
- Open source and free platform
- Active since 2007
- Global community of developers

## Extension Types

### Components
Core functionality extensions that appear in the admin menu.

### Modules
Smaller, self-contained content blocks for page positions.

### Plugins
Event-triggered extensions that extend core functionality.

### Templates
Design and layout themes for Joomla sites.

### Languages
Translation packs for internationalization.

## Additional Types

- Packages (bundled extensions)
- Files (standalone files)
- Libraries (shared code libraries)

## Development

Build extensions using:
- Joomla Framework
- PHP and MySQL
- Joomla MVC architecture
- Component Development Kit
- Joomla API
