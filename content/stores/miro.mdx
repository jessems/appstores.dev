## Overview

Miro Marketplace is the official platform for apps and integrations extending Miro's collaborative whiteboard workspace. With 100 million+ users and 250,000+ companies on the platform, it offers developers access to a large professional audience.

## Key Statistics

- **250+ apps** and integrations
- **100M+ users** worldwide (2024 milestone)
- **250,000+ companies** using Miro
- **8% AI-powered apps** in marketplace
- **60M+ potential users** for published apps

## Partner Geography

Top partner countries:
- United States: 52.75%
- Germany: 10.99%
- Netherlands: 5.49%

## Key Integrations

Popular integration categories:
- **Productivity**: Slack, Zoom, Microsoft Teams
- **Project Management**: Jira, Asana
- **Design**: Figma, Sketch
- **AI**: Glean, Amazon Q, Microsoft Copilot, Gemini Enterprise

## Development

Miro apps are built with:
- Miro Web SDK (JavaScript/React)
- REST API
- OAuth 2.0 authentication
- Webhooks for real-time updates
