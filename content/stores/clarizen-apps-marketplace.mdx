## Overview

Planview AdaptiveWork Apps Marketplace (formerly Clarizen) is the official integration platform for Planview AdaptiveWork enterprise project management. With 30+ apps and integrations including Salesforce, JIRA, Box, and Google, it extends project visibility for enterprise teams worldwide.

## Key Statistics

- **30+ integrations** (estimated)
- Enterprise project management platform
- REST and SOAP API support
- MuleSoft connector available

## History

- Founded as Clarizen in 2006
- Acquired by Planview
- Rebranded to Planview AdaptiveWork

## Key Integrations

- **Salesforce** - CRM integration
- **JIRA** - ALM/development
- **Box** - File storage
- **Google** - Workspace integration
- **Microsoft Teams** - Collaboration
- **SharePoint** - Document management
- **ServiceNow** - IT service management
- **Slack** - Team messaging

## Integration Features

### Microsoft Teams App
Comprehensive productivity interface for collaboration within the Microsoft Teams ecosystem.

### Slack Bot
Stay up-to-date with projects without leaving Slack.

### SharePoint Connector
Native connector eliminates "SharePoint sprawl" by bringing project visibility to documents.

## API Capabilities

- REST API for modern integrations
- SOAP API for legacy systems
- MuleSoft connector for enterprise integration
- Robust email engine for automation
