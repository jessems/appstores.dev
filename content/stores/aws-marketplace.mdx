## Overview

AWS Marketplace launched in 2012 and has grown to host over 14,000 software products from thousands of ISVs. It's the leading cloud marketplace for enterprise software procurement.

## Fee Structure (Updated January 2024)

| Product Type | Listing Fee |
|--------------|-------------|
| SaaS / Data Exchange | 3% |
| Server-based (AMI, containers) | 20% |
| Private offers <$1M | 3% |
| Private offers $1M-$10M | 2% |
| Private offers >$10M | 1.5% |
| Renewals | 1.5% |

**Note:** Fees only apply on transactions. No upfront listing fees.

## Key Features

### Private Offers
- Custom pricing for enterprise deals
- Negotiable terms and conditions
- Lower fees for larger deals

### AWS Consumption Integration
- Customers can apply existing AWS commitments
- Simplified procurement through AWS billing
