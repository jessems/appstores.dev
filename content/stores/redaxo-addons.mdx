## Overview

REDAXO Addons is the official addon repository for REDAXO PHP-based CMS. With 170+ addons available via the REDAXO Installer, it serves the REDAXO community with extensions for content management since 2004.

## Key Statistics

- **170+ addons** available
- PHP-based CMS since 2004
- Simple and flexible architecture
- Active German community

## Addon Types

### Core Addons
Included at installation:
- Basic CMS functionality
- User management
- Media pool

### Community Addons
- Calendar management
- News systems
- Form builders
- Backend tools
- URL schemas

## Popular Addons

- search_it (full-text search)
- symfony_mailer (email with Symfony Mailer)
- Forcal, Developer, Theme
- Focuspoint, MForm
- yCom (registration/login)
- yForm (forms)

## FriendsOfREDAXO

Community initiative to:
- Maintain orphaned addons
- Develop new extensions
- Collaborate on GitHub
- Support developers

## Development

Build addons using:
- PHP programming
- REDAXO framework
- Community documentation
- GitHub collaboration
