## Overview

Zapier Agents are AI-powered teammates that go beyond traditional chatbots by working autonomously across 8,000+ connected applications. Unlike chat-based agents that only respond when prompted, Zapier Agents proactively monitor for triggers and take action automatically.

## Agent Capabilities

### Autonomous Operation
Agents can:
- Monitor triggers across connected apps
- Make decisions based on context
- Execute multi-step workflows
- Operate 24/7 without prompting

### App Ecosystem
Access to 8,000+ integrations including:
- CRM systems (Salesforce, HubSpot)
- Communication (Slack, Email, SMS)
- Productivity (Google Workspace, Notion)
- Marketing (Mailchimp, LinkedIn)
- Development (GitHub, Jira)
- And thousands more

## Agent Types

### Lead Research Agent
- Automatically research new leads
- Enrich contact information
- Score and qualify prospects
- Sync to CRM

### IT Helpdesk Agent
- Monitor Slack channels for questions
- Provide timely responses
- Escalate complex issues
- Track resolution metrics

### Customer Support Agent
- Draft replies using knowledge base
- Handle routine inquiries
- Route to human agents as needed
- Update ticket status

### Research Agent
- Compile information from multiple sources
- Summarize findings
- Generate reports
- Stay current on topics

## 2025 Features

### Version Management
- Draft and published versions (beta)
- Rollback to previous working versions
- Safe testing before deployment

### Tool Bundle Sharing
- Create tool bundles once
- Share via link with teammates
- Each user connects own accounts
- Credentials stay private

### MCP Support
Model Context Protocol integration for:
- Extended capabilities
- Custom tool connections
- External service integration

## Creating Agents

### No-Code Builder
1. Describe what you want the agent to do
2. Connect relevant apps and data sources
3. Define triggers and conditions
4. Set oversight level (autonomous vs. approval-required)
5. Test and refine
6. Deploy

### Customization Options
- Custom instructions and personality
- Knowledge base connections
- Approval workflows
- Human-in-the-loop checkpoints

## Pricing

Zapier Agents usage counts against plan task limits:
- **Free**: Limited tasks included
- **Starter**: $19.99/month (basic agents)
- **Professional**: $49/month (advanced features)
- **Team**: $69/month (collaboration features)
- **Enterprise**: Custom pricing

## Human Oversight

Control automation levels:
- **Full Autonomy**: Agent acts independently
- **Approval Required**: Human approves actions
- **Suggestion Mode**: Agent recommends, human executes
- **Custom Rules**: Define specific oversight triggers
