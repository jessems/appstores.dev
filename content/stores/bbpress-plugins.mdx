## Overview

bbPress Plugins is the official plugin directory for bbPress WordPress forum software. With 860+ plugins on WordPress.org and 210+ curated plugins on bbPress.org, it extends bbPress forums with features like shoutboxes, attachments, notifications, and styling for 300,000+ active installations.

## Key Statistics

- **868 plugins** tagged on WordPress.org
- **210+ curated plugins** on bbPress.org
- **300,000+ active installations** of bbPress
- Tested up to WordPress 6.9

## Current Version

- **bbPress 2.6.14** (July 2025)
- 100K+ active installations
- Stewarded by Automattic since 2004
- Powers WordPress.org support forums

## History

bbPress was started in 2004 by Matt Mullenweg as a solution for WordPress support forums. It still powers the official support forums at WordPress.org.

## Popular Plugins

### Styling & Display
- **bbp style pack** - Style bbPress and add display features
- Vertical lists, hiding counts, forum styling

### File Management
- **GD bbPress Attachment** - File upload for topics and replies
- Set file limits per user

### Notifications
- **bbPress Notify** - Customize notifications (HTML, plain text, multipart)

## Plugin Categories

- Styling & themes
- Attachments & media
- Notifications & email
- User profiles
- Moderation tools
- Spam prevention
- Integration plugins

## Why 100% Free

The core bbPress plugin is completely free with no premium plans. Its popularity has spawned a large ecosystem of third-party extensions and themes, many of which are also free.
